## 已知问题

1. **MCP 服务器需要重启**: 新增的 v1.2.0 工具（wait_for_pattern, wait_for_result, get_terminal_status, resume_terminal）和模式（content_only, last_response）需要重启 MCP 服务器加载新构建
2. **Windows kill_terminal 限制**: node-pty 在 Windows 上不支持信号传递，无法通过 kill_terminal 终止进程
3. **权限提示处理**: Claude Code CLI 的权限选择菜单（1.Yes/2.Yes all/3.No）需要更精确的键盘输入（方向键选择+回车），简单的文本输入无法正确选择
